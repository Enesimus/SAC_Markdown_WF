# Caso ejemplo – UCI Pediátrica

## Objetivo

Este caso demuestra el uso completo del workflow SAC Markdown en un paciente con evolución compleja.

## Componentes del ejemplo

- evolucion.sac.md → redacción clínica en SAC Markdown
- evolucion.html → resultado compilado listo para SAC
- timeline.mmd → representación temporal del caso

## Flujo aplicado

SAC Markdown → compilación → HTML → registro clínico

## Contexto clínico

Paciente pediátrico con insuficiencia respiratoria grave que requiere ventilación mecánica, soporte hemodinámico y tratamiento antibiótico.

## Qué observar

- estructura por sistemas
- uso de macros
- segmentación clínica
- claridad narrativa
